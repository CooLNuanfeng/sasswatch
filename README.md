#### 安装

    npm install -g sasswatch



#### 运行

    sass


#### 参数说明


    -V, --version              版本号
    -e, --entry [path]         入口文件路径
    -o, --output [path]        编译输出路径
    -s, --output-style [type]  编译形式 compressed|compact|nested|expanded 默认 compressed
    -h, --help                 帮助信息
